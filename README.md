# YudzBot
YudzBot for Photop is a cool bot with many commands to use. make sure to go the the shell of your replit and run "npm i photop-client@latest", and go the the ".replit" file and delete everything and type "run = "node ."". The bot is here: https://replit.com/@YudzDoge/YudzBot, click run and go to https://app.photop.live to start using it!
